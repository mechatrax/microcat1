# MicroCat.1

MicroCat.1（マイクロキャットワン）は、MicroPython を搭載した LTE Cat.1 bis 通信モジュール一体基板です。  
このリポジトリでは MicroCat.1 のファームウェアや技術資料を公開しています。  
詳細はリンク先のページを参照してください。
